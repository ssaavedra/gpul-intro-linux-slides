Como se fai Software?

Xa sabemos como se fai hardware, máis ou menos: cóllenese enxeñeiros eléctricos e físicos, deséñanse circuitos, imprímense e mecanízanse, pero e o software?
Ídelo ver e realizar nos próximos anos.

  • Escríbese código fonte nalgunha linguaxe de programación
    (que típicamente non é Pascal)
  • Un compilador (e un enlazador) conséguennos un ficheiro executable
  • O ficheiro executable é comprendido pola máquina

Que é o Software Libre?

Libre de liberdade.
  1. Uso con calquera finalidade
  2. Estudo e modificación. Acceso á fonte (open source).
  3. Distribuir copias
  4. Modificar o programa e distribuir as modificacións

Que é GNU?

GNU is Not Unix

Un proyecto para realizar Software Libre, co obxectivo de crear un sistema operativo completamente libre: o Sistema GNU.
